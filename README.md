# ElvUI BagCompanion (Ascension WoW)

**ElvUI BagCompanion** is a lightweight, feature-rich addon designed for the [Ascension WoW](https://ascension.gg/) client. It provides a convenient toolbar for managing your inventory, banking, and vendor interactions directly within the ElvUI interface.

### Features
*   **Bank Management:** Quickly deposit or withdraw items by quality (Poor, Common, Uncommon, Rare, Epic, Legendary, Vanity) or by category (Herbs, Leather, Ore, Cooking, Cloth).
*   **Quick Sell:** Automatically sell green/blue equipment at vendors with a single click.
*   **Junk Cleanup:** Easily delete gray (junk) and useless white items with an optional confirmation prompt.
*   **Transmog Collection:** One-click utility to collect appearances from all items in your bags.
*   **Integrated UI:** Adds a sleek, animatable toolbar above your bags that integrates seamlessly with ElvUI’s style.
*   **Customizable Settings:** Toggle which quality tiers you want to handle and enable/disable deletion confirmations.

# ElvUI BagCompanion (Ascension WoW)

**ElvUI BagCompanion** — это легкий и удобный аддон для клиента [Ascension WoW](https://ascension.gg/), предоставляющий панель инструментов для управления инвентарем, банком и торговлей прямо в интерфейсе ElvUI.

### Возможности
*   **Управление банком:** Быстрая выгрузка или забор предметов из банка по их качеству или категории (травы, кожа, руда, кулинария, ткань).
*   **Быстрая продажа:** Автоматическая продажа "зеленой" и "синей" экипировки вендору одним нажатием.
*   **Очистка хлама:** Удобное удаление "серых" и бесполезных белых предметов с настраиваемым подтверждением.
*   **Коллекционирование:** Кнопка для быстрого добавления всех предметов из сумок в коллекцию трансмогрификации (Transmog Collection).
*   **Интеграция с ElvUI:** Стильная анимированная панель инструментов, которая появляется над сумками и идеально вписывается в дизайн ElvUI.
*   **Настройки:** Возможность выбирать категории качества для обработки и настраивать подтверждение удаления предметов.

### Installation / Установка
1. Download the source code as a ZIP.
2. Unpack the folder into your `World of Warcraft/Interface/AddOns/` directory.
3. Rename the folder to `ElvUI_BagCompanion` (if necessary).
4. Log in and enjoy!
